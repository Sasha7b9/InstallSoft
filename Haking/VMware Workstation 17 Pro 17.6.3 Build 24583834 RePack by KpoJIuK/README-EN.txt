Общая информация:

Тип: установка
Языки: английский
Лечение: программа бесплатна

Дополнительно: применён патч, разблокирующий установку MacOS
Для запуска MacOS в VMX-файл машины необходимо добавить строку
smc.version = "0"

В сборку встроены только VMware Tools для Windows и MacOS.
Остальные программа загружает из сети автоматически.

Параметры командной строки:

Тихая установка английской версии: /S /QE
Выбор места установки: /D=ПУТЬ
Параметр /D=ПУТЬ необходимо указывать самым последним
Например: Install.exe /S /QE /D=C:\MyProgram
